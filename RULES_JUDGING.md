# 🏆 Hackathon Rules & Judging Criteria

## Eligibility & Rules

### Team Rules
- **Team Size**: 1-4 people per team
- **Registration**: Submit team info via GitHub issue
- **Commitment**: Teams commit to finishing in designated timeframe

### Code Submission
- **Repository**: Must be public GitHub repository
- **Language**: Python strongly recommended
- **Documentation**: Include README.md explaining your project
- **Dependencies**: List all external libraries (requirements.txt)

### Guidelines
✅ **Allowed**
- External libraries (NumPy, PyTorch, etc.)
- Pre-trained models
- Starter code from OpenEnv examples
- Collaboration and code review
- Open source projects

❌ **Not Allowed**
- Plagiarism or copying previous submissions
- Proprietary/closed-source code
- External API keys in repository
- Code written before hackathon

## Judging Criteria

### Scoring Breakdown (Total: 100 points)

#### 1. Innovation & Creativity (30 points)
- Novel approach to agent design
- Unique use of OpenEnv features
- Creative problem-solving

#### 2. Technical Execution (30 points)
- Code quality and organization
- Correctness of implementation
- Efficiency and optimization
- Testing and error handling

#### 3. OpenEnv Alignment (20 points)
- Proper use of OpenEnv API
- Demonstrates understanding of framework
- Appropriate environment selection

#### 4. Presentation & Documentation (20 points)
- Clear README and documentation
- Well-explained approach
- Demo quality

## Submission Requirements

### Required Files
```
your-repo/
├── README.md                 # Project overview & setup
├── requirements.txt          # Dependencies
├── src/                      # Main code
│   └── agent.py             # Your agent implementation
└── examples/                 # Usage examples
    └── run_agent.py         # Demo script
```

## Common Disqualifications

⚠️ **Projects will be disqualified for:**
- Plagiarism (copying others' work)
- Code written before hackathon
- Offensive or harmful content
- Violating OpenEnv license (BSD-3-Clause)
- Missing core requirement files
- Broken code that won't run

Good luck! 🚀
